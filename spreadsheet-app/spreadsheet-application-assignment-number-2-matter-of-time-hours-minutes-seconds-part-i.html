Spreadsheet Application Assignment 1. A Matter of Time  (Hours, Minutes, Seconds) Part I

Background (How Dates are represented in Excel and in JavaScript): Dates and times are often represented internally as numbers. You can see this if you enter a number in Excel and then change its display to a Date/Time format.  In Excel, Dates are represented as the number of days since December 31, 1899 at midnight (represented by 0; 1 represents a day later or January 1, 1900 at midnight).  Fractions of days affect the hours so e.g., 3.25 represents January 3, 1900 at 6am (3 and 1 quarter day after December 31 at midnight).  From https://support.microsoft.com/en-us/help/214094/how-to-use-dates-and-times-in-excel :

"In this system, the serial number 1 represents 1/1/1900 12:00:00 a.m. Times are stored as decimal numbers between .0 and .99999, where .0 is 00:00:00 and .99999 is 23:59:59. The date integers and time decimal fractions can be combined to create numbers that have a decimal and an integer portion. For example, the number 32331.06 represents the date and time 7/7/1988 1:26:24 a.m."

In JavaScript, Dates are represented by the number of milliseconds since January 1, 1970 at midnight UTC (Coordinated Universal Time), where 0 represents this time.  If you enter console.log(new Date(0)); in the Eastern US, you will get "Wed Dec 31 1969 19:00:00 GMT-0500 (EST)."  This is because Eastern Standard Time is 5 hours behind UTC (represented by the -0500).  Note that there are 1000 milliseconds (ms) in a second so if you enter console.log(new Date(4000)); in the Eastern US, you will get "Wed Dec 31 1969 19:00:04 GMT-0500 (EST)." (4 seconds later)  From the documentation at https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date :

Date objects are based on a time value that is the number of milliseconds since 1 January, 1970 UTC (midnight).

Quick Overview: For this first assignment, you will write code to make it possible for a user to enter a number in a cell and then change its format to "Time (given in ms)" or "Time (given in days)", whereby the cell will display the corresponding time.  (Days and dates will be handled in a later assignment.)

  2. Define a function convertToLarger that takes two inputs, a number and a positive integer conversion factor, and returns the total number of whole units in the number with that conversion factor. For example, if you wanted to convert 30 quarters to dollars, you could call convertToLarger(30, 4), reflecting that there are 4 quarters in a dollar and get 7 as an output, the number of whole dollars you could get from 30 quarters.  So the conversion factor here is 4.  Note that you could also use convertToLarger(30, 4) to convert from 30 quarts to gallons since the conversion factor is the same (4 quarts in a gallon).

Using msToTotalMinutes and remainingAfterConvert, define a function getMinuteFromMs that takes the given number of milliseconds from midnight and returns the minute (0 to 60) for the displayed time.  Note: This function is already present in the code; remove the 0 and replace it with the appropriate expression.
  9. Using msToTotalHours and remainingAfterConvert, define a function getHourFromMs that takes the given number of milliseconds from midnight and returns the hour (0 to 24) for the displayed time.  Note: This function is already present in the code; remove the 0 and replace it with the appropriate expression.
