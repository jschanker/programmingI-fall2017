Strings

Characters and Strings

  • A character is a single letter, number, symbol, unit of space, etc.
  • A string is a series of 0 or more characters.  A string of length 0 is called the empty string.
  • A string literal is a particular string.
    • Example: "Hey Hey Hey!" is a string literal that's 12 characters in length.
  • As with numbers, strings can be assigned to variables.  They can also be the inputs to or the outputs from a function. 
    let greeting = "Hey";
let double = s => s + s;
console.log(double(greeting)); // outputs HeyHey

Length of strings/Accessing Individual Characters/Properties & Methods

  • If s is a string, you can get its length by using s.length. Here, length refers to a property of the string s.  A property is like a variable but it's connected to a specific object, which in this case is the string s: 
    console.log("Hey Hey Hey!".length); // prints 12, the length of the string literal, "Hey Hey Hey!"
  • Individual characters of a string can be accessed using the charAt method of strings. In JavaScript, a method is a property of an object that happens to be a function. The function takes an implicit parameter as one of its inputs, which in this case is the string itself.  Methods define what an object can do.  Just as with general functions that make sense for some types of inputs and not for others, methods are specific to certain types of objects. 
    charAt = (s, index) => s.charAt(index);
console.log("JavaScript is awesome!".charAt(2));
console.log(charAt("JavaScript is awesome!", 2)); // same as above
console.log(3.14.charAt(2)); // TypeError: 3.14.charAt is not a function 
                             // (charAt is not a method of numbers) 
console.log(charAt(3.14, 2)); // TypeError: s.charAt is not a function 
                              // (charAt takes a string and number as inputs)
  • The charAt method takes a nonnegative integer as an explicit input and the string as an implicit one, and returns the character at that position.  The positions (or indices as they're called) start with 0 so the first letter is at index 0, the second letter is at index 1, the third letter at index 2, etc. 
    let getFifthChar = phrase => phrase.charAt(4);
console.log(getFifthChar("HEY KIND WORLD!")); // prints K
//                        01234
console.log(getFifthChar("STAY KIND WORLD!")); // prints space
//                        01234
  • Note that the charAt method must be called on a string and take a nonnegative integer that's less than the length of the string as an input. It returns a character as an output.

Concatenation

You can join together multiple strings (includes single letters) by using the '+' operator.

let makeGreeting = name => "Hey there, " + name + "!";
console.log(makeGreeting("Alice")); // logs Hey there, Alice!

Get substring of string from start to end position

If s is a string, s.substring(i, j) is the substring of characters of s starting at index i and ending at index j-1 (before j). If j is omitted, the substring is from i to the end.

console.log("Hey Hey Hey!".substring(4, 10)); // logs "Hey He"
//           0123456789
console.log("Molloy123!!!".substring(4,10));  // logs "oy123!"
//           0123456789
console.log("Molloy123!!!".substring(4,10));  // logs "oy123!!!"
//           01234

Find Start Position of Substring in String

If s is a string, s.indexOf(str, [startPos=0]) finds the index of the first occurrence of str starting at position startPos (optional parameter) or starting at the beginning of the string if startPos is not given.  -1 is returned if the string isn’t located at or after position startPos.

let getFifthChar = phrase => phrase.charAt(4);
console.log("Hey Hey Hey!".indexOf("ey")); // prints 1
//           012
console.log("Hey Hey Hey!".indexOf("ey",2)); // prints 5 (starts looking at index 2)
//           0123456
console.log("Hey Hey Hey!".indexOf("ey",9)); // prints 9 (find ey at starting index)
//           0123456789
console.log("Hey Hey Hey!".indexOf("ey",10)); // prints -1 (doesn't find ey starting at index 10)
//           0123456789
