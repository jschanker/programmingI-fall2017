Arrays

Arrays are lists of items. Arrays can be assigned to variables as well. Here are some examples:

let numberList = [1,5,7];
let fruitList = ["apple", "banana", "cantaloupe", "orange"];

Arrays can also be passed to functions or returned from functions.

let firstItem = arr => arr[0];
console.log(firstItem(fruitList)); // logs apple
let getItems = itemList => itemList.split(",");
console.log(getItems("apple,banana,cantaloupe,orange")); // logs [ 'apple', 'banana', 'cantaloupe', 'orange' ]

Notice how we use [...] replaced with the index of the item we want to get the item at that index so [0] is used to get the first item, [1] is used to get the second item, [2] is used to get the third item, etc.  The split method on strings can be used to create an array from a string by using the inputted character(s) as separators for each item. In the example above, the separator is the , and an array of four items is produced.

Solving Repeat for Every Item In Array, Finding the First In Array Problems by Forward Recursion

For finding the first, the setup will look something like this (matching the table):

let funcName = (itemList,additional variables) => {
   let condition = item => _________________________;
   let makeTable = (index, item) => {
     if(condition(item)) return item;
     else return makeTable(index+1, itemList[index+1]);
   };
   return makeTable(0, itemList[0]);
};

So to find the first positive number in the list, we'd have the following (may want to replace itemList with numList and item with num):

let firstPositive = (itemList) => {
   let isPositive = item => item > 0;
   let makeTable = (index, item) => {
     if(isPositive(item)) return item;
     else return makeTable(index+1, itemList[index+1]);
   };
   return makeTable(0, itemList[0]);
};

console.log(firstPositive([-7,0,-3,4,8,-5,10])); // logs 42

For applying an operation to every element in the list, the setup will look something like this (matching the table):

let funcName = (itemList,additional variables) => {
   let operation = (acc, item) => _________________________;
   let makeTable = (index, acc) => {
     if(index === itemList.length-1) return acc;
     else return makeTable(index+1, operation(acc, itemList[index+1]));
   };
   return makeTable(0, itemList[0]);
};

The length property of Arrays refers to the number of elements in the array just as the length property of strings is the number of characters in the string. If we view a string as an array of characters, these notions coincide (e.g., "abc".length is the same as ['a', 'b', 'c'],length, mainly 3). We want to stop in the row where we've performed the operation on the last item in the array at index itemList.length-1 so we stop at this index.

Example (Sum all the numbers in a list):

let sumAllNumbersInArray = itemList => {
   let plus = (acc, item) => acc + item;
   let makeTable = (index, acc) => {
     if(index === itemList.length-1) return acc;
     else return makeTable(index+1, plus(acc, itemList[index+1]));
   };
   return makeTable(0, itemList[0]);
};

console.log(sumAllNumbersInArray([1,2,3,4,5])); // logs 15

Alternatively, we may want to start acc with an initial value startValue instead of the 0th item in an array. In that case, we could add a "dummy starting row" with -1 as the input and startValue as the output:

let funcName = (itemList,additional variables) => {
   let operation = (acc, item) => _________________________;
   let makeTable = (index, acc) => {
     if(index === itemList.length-1) return acc;
     else return makeTable(index+1, operation(acc, itemList[index+1]));
   };
   return makeTable(-1, startValue);
};

In the case of the sum example, we could also start the sum at 0 so we'd have the following:

let sumAllNumbersInArray2 = itemList => {
   let plus = (acc, item) => acc + item;
   let makeTable = (index, acc) => {
     if(index === itemList.length-1) return acc;
     else return makeTable(index+1, plus(acc, itemList[index+1]));
   };
   return makeTable(-1, 0);
};

console.log(sumAllNumbersInArray2([1,2,3,4,5])); // logs 15
